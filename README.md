# 2026 월드컵 예측 게임

## 1. 이벤트 개요

팀 워크샵에서 함께 월드컵 경기를 관람하며, 친목을 위한 **월드컵 예측 게임**을 진행합니다.

실제 금전 베팅이 아닌, 경기 결과를 예측하고 점수를 합산하는 **사내 레크리에이션 이벤트**입니다.

| 구분 | 내용 |
|---|---|
| 앱 주소 | https://worldcup-score-plan.vercel.app |
| 참여 방법 | 이름·사번·전화번호로 직원 인증 후 로그인 |
| 대상 경기 | 1차전: 대한민국 vs 체코 / 2차전: 대한민국 vs 멕시코 |
| 진행 방식 | 경기 전 1차 예측 → 하프타임 일부 수정 → 경기 후 MVP 투표 → 최종 점수 공개 |
| 목적 | 팀 친목, 응원 분위기 조성, 경기 관람 재미 확대 |

---

## 2. 경기 일정

| 차수 | 경기 | 일정 (KST) |
|---|---|---|
| 1차전 | 🇰🇷 대한민국 vs 체코 🇨🇿 | 2026년 6월 15일 |
| 2차전 | 🇰🇷 대한민국 vs 멕시코 🇲🇽 | 2026년 6월 19일 |

각 경기는 독립적으로 진행되며, 차수별로 예측을 따로 제출합니다.

---

## 3. 참여 방법

1. 앱 주소(https://worldcup-score-plan.vercel.app)에 접속합니다.
2. **로그인** 버튼을 눌러 이름·사번·전화번호를 입력하고 인증합니다.
3. 상단 탭에서 참여할 경기(1차전 / 2차전)를 선택합니다.
4. 경기 시작 전까지 예측 항목을 모두 입력하고 제출합니다.

---

## 4. 전체 진행 흐름

```
경기 시작 전   →   전반 종료   →   하프타임   →   경기 종료   →   결과 공개
1차 예측 제출       중간 현황       일부 수정       MVP 투표        최종 순위 발표
```

1. 경기 시작 전까지 예측 항목을 제출합니다.
2. 전반 종료 후 운영자가 전반 결과를 입력하면 하프타임 수정 화면이 열립니다.
3. 하프타임에 일부 항목을 1회 수정할 수 있습니다.
4. 경기 종료 후 MVP를 최종 투표합니다.
5. 운영자가 최종 결과를 입력하면 점수가 자동 계산됩니다.
6. 운영자가 결과를 공개하면 최종 순위를 확인할 수 있습니다.

## 5. 예측 항목

| 번호 | 예측 항목 | 예시 | 하프타임 수정 가능 여부 |
|---:|---|---|---|
| 1 | 경기 결과 | 대한민국 승 / 무승부 / 상대팀 승 | 가능 |
| 2 | 최종 스코어 | 대한민국 2 : 1 상대팀 | 가능 |
| 3 | 양 팀 총 득점 수 | 총 3골 | 가능 |
| 4 | 대한민국 첫 득점자 | 손흥민, 이강인, 없음 등 | 전반 무득점 시만 가능 |
| 5 | 첫 골 팀 | 대한민국 / 상대팀 / 없음 | 전반 무득점 시만 가능 |
| 6 | 첫 골 시간대 | 전반 0~15분, 16~30분 등 | 전반 무득점 시만 가능 |
| 7 | 전반전 결과 | 대한민국 리드 / 무승부 / 상대팀 리드 | **수정 불가** |
| 8 | 양 팀 카드 수 | 0~2장 / 3~5장 / 6장 이상 | 가능 |
| 9 | MVP | 경기에서 가장 활약한 선수 | 경기 후 최종 1회 투표 |

## 7. 하프타임 수정 규칙

### 수정 가능한 항목

| 항목 | 수정 가능 기준 |
|---|---|
| 경기 결과 | 항상 수정 가능 |
| 최종 스코어 | 항상 수정 가능 |
| 양 팀 총 득점 수 | 항상 수정 가능 |
| 양 팀 카드 수 | 항상 수정 가능 |
| 대한민국 첫 득점자 | 전반에 대한민국 득점이 없었을 경우만 가능 |
| 첫 골 팀 | 전반에 골이 없었을 경우만 가능 |
| 첫 골 시간대 | 전반에 골이 없었을 경우만 가능 |

### 수정 불가능한 항목

- **전반전 결과**: 전반 종료 시점에 이미 확정
- **전반에 이미 발생한 이벤트**: 첫 골 팀·시간대·대한민국 첫 득점자

### 수정 시 주의사항

- 하프타임 수정은 **1회만 가능**합니다.
- 수정하지 않으면 경기 전 1차 예측값이 그대로 사용됩니다.
- 경기 결과·스코어·총 득점 수는 서로 논리적으로 일치해야 합니다.
  - 예: 스코어를 2:1로 수정했다면 경기 결과도 반드시 대한민국 승으로 수정해야 합니다.

## 8. MVP 투표 방식

MVP는 **참여자 전원이 투표**하여 결정합니다.

1. 경기 시작 전에 1차 MVP를 선택합니다.
2. 경기 종료 후 MVP를 최종 1회 수정할 수 있습니다. 미수정 시 1차 예측값이 그대로 사용됩니다.
3. **모든 참여자의 투표를 집계하여 가장 많은 표를 받은 선수가 MVP**로 결정됩니다.
4. MVP로 결정된 선수를 예측한 참여자가 5점을 획득합니다.
5. 동표인 경우 이름 가나다순 앞쪽 선수가 MVP로 결정됩니다.

| 구분 | 기준 | 점수 |
|---|---|---:|
| MVP 맞힘 | 최다득표 선수를 예측 | 5점 |
| MVP 틀림 | 최다득표 선수와 다른 선수 예측 | 0점 |
| 최종 미제출 | 1차 예측값으로 자동 채점 | 최대 5점 |

---

## 9. 동점자 처리 기준

동점자가 있을 경우 아래 순서로 순위를 결정합니다.

| 순서 | 기준 |
|---|---|
| 1 | 정확한 스코어를 맞힌 사람 우선 |
| 2 | 경기 결과를 맞힌 사람 우선 |
| 3 | 대한민국 첫 득점자를 맞힌 사람 우선 |
| 4 | 첫 골 시간대를 맞힌 사람 우선 |
| 5 | 하프타임 수정을 사용하지 않은 사람 우선 |
| 6 | 예측을 먼저 제출한 사람 우선 |

## 10. 예측 예시

### 경기 전 1차 예측

| 항목 | 예측 예시 |
|---|---|
| 경기 결과 | 대한민국 승 |
| 최종 스코어 | 대한민국 2 : 1 멕시코 |
| 양 팀 총 득점 | 3골 |
| 대한민국 첫 득점자 | 손흥민 |
| 첫 골 팀 | 대한민국 |
| 첫 골 시간대 | 전반 16~30분 |
| 전반전 결과 | 무승부 |
| 양 팀 카드 수 | 3~5장 |
| MVP 1차 예측 | 이강인 |

### 하프타임 수정 예시

전반 종료 스코어가 대한민국 0 : 1 멕시코인 경우:

| 항목 | 경기 전 예측 | 하프타임 수정 |
|---|---|---|
| 경기 결과 | 대한민국 승 | 무승부 |
| 최종 스코어 | 대한민국 2 : 1 멕시코 | 대한민국 1 : 1 멕시코 |
| 양 팀 총 득점 | 3골 | 2골 |
| 양 팀 카드 수 | 3~5장 | 0~2장 |

> 전반에 이미 멕시코가 첫 골을 넣었으므로 `첫 골 팀`과 `첫 골 시간대`는 수정할 수 없습니다.

## 11. 운영 유의사항

- 본 이벤트는 팀 친목을 위한 레크리에이션입니다. 실제 금전 베팅은 진행하지 않습니다.
- 경기 시작 후에는 1차 예측을 제출할 수 없습니다.
- 전반전 결과 항목은 경기 전 제출 후 수정이 불가합니다.
- 하프타임 수정은 1회이며 마감 이후에는 MVP를 제외한 항목을 수정할 수 없습니다.
- MVP 최종 투표는 경기 종료 후 운영자가 안내하는 시간 내에 제출해야 합니다.
- 경기 기록은 최종 공식 경기 결과를 기준으로 합니다.

## 12. 운영자 진행 안내

### 경기 시작 전

> 지금부터 예측 게임을 시작합니다. 경기 시작 전까지 모든 항목을 입력해 주세요.  
> 전반 종료 후 하프타임에 일부 항목을 1회 수정할 수 있습니다.

### 전반 종료 후

> 전반전이 종료되었습니다. 전반 결과를 입력하면 하프타임 수정 화면이 열립니다.  
> 수정 가능 항목을 확인하고 후반 예측에 맞게 수정해 주세요.

### 경기 종료 후

> 경기가 종료되었습니다. MVP 최종 투표를 진행해 주세요.  
> 투표 마감 후 최종 결과를 입력하고 순위를 공개하겠습니다.
