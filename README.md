# 대한민국 vs 멕시코 월드컵 예측 게임 안내

## 1. 이벤트 개요

팀 워크샵에서 함께 월드컵 경기를 관람하며, 친목을 위한 **월드컵 예측 게임**을 진행합니다.

이번 게임은 실제 금전 베팅이 아닌, 경기 결과를 예측하고 점수를 합산하는 **사내 레크리에이션 이벤트**입니다.  
경기 지식이 많지 않아도 누구나 쉽게 참여할 수 있으며, 경기 관람의 몰입감과 재미를 높이는 것을 목적으로 합니다.

| 구분 | 내용 |
|---|---|
| 경기 | 대한민국 vs 멕시코 |
| 일정 | 2026년 6월 19일 |
| 방식 | 경기 전 1차 예측 제출, 전반 종료 후 중간 점수 집계, 하프타임 일부 항목 수정, 경기 종료 후 최종 점수 합산 |
| 선정 | 총점 기준 1등, 2등, 3등 선정 |
| 목적 | 팀 친목, 응원 분위기 조성, 경기 관람 재미 확대 |

---

## 2. 전체 진행 방식

이번 예측 게임은 **경기 전 예측, 전반 종료 후 중간 집계, 하프타임 수정, 경기 종료 후 최종 집계** 순서로 진행합니다.

1. 경기 시작 전까지 1차 예측 항목을 모두 제출합니다.
2. 전반 종료 후, 전반 기준으로 확정 가능한 항목의 점수를 중간 집계합니다.
3. 하프타임에 일부 항목만 수정 제출할 수 있습니다.
4. 이미 전반에 확정된 항목은 수정할 수 없습니다.
5. 경기 종료 후 최종 경기 결과와 하프타임 수정 제출 내용을 반영하여 최종 점수를 계산합니다.
6. 총점이 높은 순서대로 1등, 2등, 3등을 선정합니다.
7. 동점자가 있을 경우 별도 동점자 기준에 따라 순위를 결정합니다.

## 4. 하프타임 수정 규칙

하프타임에는 경기 흐름을 반영하여 일부 항목을 수정할 수 있습니다.

### 4.1 수정 가능한 항목

| 항목 | 수정 가능 기준 |
|---|---|
| 경기 결과 | 하프타임에 수정 가능 |
| 최종 스코어 | 하프타임에 수정 가능 |
| 양 팀 총 득점 수 | 하프타임에 수정 가능 |
| 양 팀 카드 수 | 하프타임에 수정 가능 |
| 대한민국 첫 득점자 | 전반에 대한민국 득점이 없었을 경우에만 수정 가능 |
| 첫 골 팀 | 전반에 골이 없었을 경우에만 수정 가능 |
| 첫 골 시간대 | 전반에 골이 없었을 경우에만 수정 가능 |
| MVP | 경기 종료 후 최종 1회 수정 가능 |

### 4.2 수정 불가능한 항목

| 항목 | 수정 불가 사유 |
|---|---|
| 전반전 결과 | 전반 종료 시점에 이미 확정된 항목 |
| 전반에 이미 발생한 첫 골 팀 | 경기 내 첫 골은 이미 확정된 항목 |
| 전반에 이미 발생한 첫 골 시간대 | 경기 내 첫 골 시간대는 이미 확정된 항목 |
| 전반에 이미 발생한 대한민국 첫 득점자 | 대한민국 첫 득점자는 이미 확정된 항목 |

### 4.3 하프타임 수정 제출 기준

- 하프타임 수정은 **1회만 가능**합니다.
- 하프타임 종료 전까지 제출해야 합니다.
- 수정하지 않은 항목은 경기 전 1차 예측값을 그대로 사용합니다.
- 하프타임 수정 마감 이후에는 MVP를 제외한 항목을 다시 수정할 수 없습니다.
- 경기 결과, 최종 스코어, 총 득점 수는 서로 논리적으로 일치해야 합니다.
  - 예: 최종 스코어를 대한민국 2 : 1 멕시코로 수정했다면 경기 결과는 대한민국 승이어야 합니다.

## 6. 전반 종료 후 중간 집계

전반전이 종료되면 운영자는 전반 기준으로 확정 가능한 항목을 먼저 집계합니다.

### 6.1 중간 집계 대상 항목

| 항목 | 배점 | 중간 집계 가능 여부 |
|---|---:|---|
| 전반전 결과 | 5점 | 가능 |
| 첫 골 팀 | 5점 | 전반에 골이 나온 경우 가능 |
| 첫 골 시간대 | 10점 | 전반에 골이 나온 경우 가능 |
| 대한민국 첫 득점자 | 15점 | 전반에 대한민국 득점이 나온 경우 가능 |
| 카드 수 | 5점 | 중간 현황만 공유, 최종 채점은 경기 종료 후 |

### 6.2 중간 집계 운영 방식

1. 전반 종료 후 운영자가 전반 결과와 발생 이벤트를 확인합니다.
2. 확정된 항목 기준으로 임시 점수를 계산합니다.
3. 하프타임에 현재 순위 또는 상위권 현황을 공유합니다.
4. 중간 점수는 최종 점수가 아니며, 후반 결과에 따라 순위가 바뀔 수 있습니다.
5. 중간 집계 후 하프타임 수정 가능 항목을 안내합니다.

### 6.3 중간 점수 공개 방식

중간 점수는 아래 방식 중 하나로 공유합니다.

| 방식 | 설명 |
|---|---|
| 전체 순위 공개 | 모든 참여자의 중간 점수와 순위를 공개 |
| Top 5만 공개 | 상위 5명만 공개하여 긴장감 유지 |
| 점수 구간 공개 | 1위권, 추격권, 역전 가능권처럼 구간으로 공개 |

추천 방식은 **Top 5만 공개**입니다.  
전체 점수를 공개하면 하위권 참여자의 몰입도가 떨어질 수 있으므로, 상위권만 공개하고 “후반 역전 가능” 분위기를 만드는 것이 좋습니다.

## 7. MVP 제출 방식

MVP는 경기 시작 전에 1차로 입력합니다.  
다만 MVP는 경기 흐름을 본 뒤 판단하는 재미 요소가 있으므로, **경기 종료 후 MVP 항목만 최종 1회 수정 제출할 수 있습니다.**

경기 후 최종 제출한 MVP가 운영진이 선정한 MVP와 일치하면 5점입니다.

MVP 선정 기준은 아래 순서로 적용합니다.

1. 공식 경기 MOM 또는 POTM이 확인 가능한 경우 공식 기준 적용
2. 공식 기준 확인이 어려운 경우 현장 투표로 MVP 선정
3. 현장 투표가 어려운 경우 운영진이 경기 기여도, 득점, 도움, 경기 영향도를 기준으로 선정

| 구분 | 기준 | 점수 |
|---|---|---:|
| 경기 전 MVP 1차 입력 | 참여 기록용, 경기 전 예측 재미 요소 | 점수 미반영 |
| 경기 후 MVP 최종 제출 | 최종 채점 기준 | 5점 |
| MVP 최종 제출 누락 | 경기 전 입력값을 최종 제출값으로 간주 | 최대 5점 |

## 12. 운영 유의사항

- 본 이벤트는 팀 친목을 위한 레크리에이션입니다.
- 실제 금전 베팅은 진행하지 않습니다.
- 경기 전 1차 예측 제출은 경기 시작 전까지 진행합니다.
- 전반전 결과 항목은 경기 시작 후 수정할 수 없습니다.
- 하프타임에는 일부 항목만 1회 수정할 수 있습니다.
- 이미 전반에 확정된 이벤트 항목은 하프타임에 수정할 수 없습니다.
- 하프타임 수정 마감 이후에는 MVP를 제외한 항목을 수정할 수 없습니다.
- MVP 항목은 경기 종료 후 최종 1회 수정 제출할 수 있습니다.
- MVP 최종 제출 마감 이후에는 수정할 수 없습니다.
- 경기 기록은 최종 공식 경기 결과 또는 운영진이 확인한 결과를 기준으로 합니다.
- 모든 참여자가 즐겁게 관람할 수 있도록 응원과 배려를 함께 부탁드립니다.

> 전반 종료 후 중간 점수를 공개하고, 하프타임에는 일부 항목을 한 번 수정할 수 있습니다.

### 전반 종료 후

> 전반전이 종료되었습니다.  
> 지금부터 전반전 결과와 확정된 이벤트 항목을 기준으로 중간 점수를 집계하겠습니다.  
> 잠시 후 하프타임 수정 가능 항목을 안내드리겠습니다.

### 하프타임 수정 안내

> 하프타임 수정이 시작되었습니다.  
> 경기 결과, 최종 스코어, 총 득점 수, 카드 수는 수정할 수 있습니다.  
> 이미 전반에 확정된 첫 골 팀, 첫 골 시간대, 대한민국 첫 득점자는 수정할 수 없습니다.  
> 수정하지 않으면 경기 전 제출값이 그대로 적용됩니다.

### 경기 종료 후

> 경기가 종료되었습니다.  
> 이제 MVP 최종 제출을 진행합니다.  
> MVP는 경기 후 최종 1회 수정 가능하며, 제출 마감 후 최종 점수를 집계해 1등부터 3등까지 발표하겠습니다.

## 14. 한 줄 요약

**경기 전 예측하고, 전반 종료 후 중간 점수를 확인한 뒤, 하프타임에 일부 항목을 수정하고, 경기 후 MVP까지 반영해 최종 1~3등을 선정하는 월드컵 친목 예측 게임입니다.**
