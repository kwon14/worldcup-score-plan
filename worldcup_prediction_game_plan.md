# 2026 월드컵 대한민국 vs 멕시코 예측 포인트 게임 구성안

## 1. 개요

본 문서는 팀 워크샵에서 2026년 6월 19일 예정된 대한민국 vs 멕시코 월드컵 경기를 함께 관람하며 진행할 수 있는 사내 친목형 예측 포인트 게임의 구성안이다.

본 이벤트는 팀원 간 소통과 응원 분위기 조성을 위한 레크리에이션 목적의 예측 게임으로 운영한다. 참여자는 경기 시작 전까지 1차 예측을 제출하고, 전반 종료 후 하프타임에 중간 점수를 확인한 뒤 일부 후반 관련 항목을 수정 제출할 수 있다. 경기 종료 후에는 MVP 항목만 최종 1회 수정 제출할 수 있으며, 시스템은 실제 경기 결과와 각 제출값을 비교하여 최종 점수를 산정한다.

본 구성안은 **A안, 실력형 순위제**를 기준으로 한다. 즉, 점수 합산 결과에 따라 1등, 2등, 3등을 선정한다.

---

## 2. 운영 목적

### 2.1 핵심 목적

- 월드컵 경기 관람을 통한 팀원 간 친목 향상
- 전반전, 하프타임, 후반전까지 자연스럽게 이어지는 참여형 이벤트 운영
- 경기 전 예측의 긴장감과 하프타임 수정의 전략성을 함께 제공
- 경기 전후 자연스러운 대화와 응원 분위기 조성
- 축구 지식이 많은 사람뿐 아니라, 모든 팀원이 쉽게 참여할 수 있는 이벤트 운영
- 워크샵 프로그램 내 몰입감과 재미 요소 강화

### 2.2 운영 방향

- 실제 금전 베팅이 아닌 포인트 기반 예측 게임으로 운영
- 경기 지식, 예측력, 현장 판단, 운 요소가 적절히 섞이도록 항목 구성
- 점수 산정 기준과 수정 가능 항목을 사전에 공개하여 공정성 확보
- 전반 종료 시점에 중간 점수를 공개하여 하프타임 몰입도 향상
- 웹페이지를 통해 예측 제출, 하프타임 수정, MVP 수정, 결과 확인, 순위 집계를 진행할 수 있도록 설계

## 3. 게임 방식

### 3.1 기본 방식

1. 참여자는 경기 시작 전까지 웹페이지에서 1차 예측 항목을 입력한다.
2. 경기 시작 후에는 전반전 관련 확정 항목과 이미 발생한 이벤트 항목을 수정할 수 없다.
3. 전반 종료 후 운영자는 전반전 실제 결과를 입력하고, 시스템은 1차 중간 점수를 집계한다.
4. 하프타임에 참여자는 일부 후반 관련 항목을 1회 수정 제출할 수 있다.
5. 후반전 시작 시 하프타임 수정 제출을 마감한다.
6. 경기 종료 후 참여자는 MVP 항목만 최종 1회 수정 제출할 수 있다.
7. 경기 종료 후 운영자가 최종 실제 경기 결과를 입력한다.
8. 시스템은 경기 전 예측값, 하프타임 수정값, 경기 후 MVP 최종 제출값을 기준으로 점수를 자동 계산한다.
9. 총점 기준으로 1등, 2등, 3등을 선정한다.

### 3.2 참여 대상

- 워크샵 참석 팀원 전체
- 축구 지식 여부와 관계없이 누구나 참여 가능

### 3.3 운영 방식

- 개인전 방식으로 운영
- 총점 100점 만점
- 경기 전 예측, 하프타임 수정, 경기 후 MVP 최종 수정의 3단계 제출 구조
- 동일 점수 발생 시 타이브레이커 기준 적용
- 최종 순위는 경기 종료 후 웹페이지에 공개

## 6. 하프타임 수정 가능 항목

하프타임 수정은 경기의 재미와 전략성을 높이기 위한 장치이다. 다만 공정성을 위해 이미 전반전에 결과가 확정된 항목은 수정할 수 없다.

### 6.1 수정 가능 항목

| 항목 | 수정 가능 여부 | 기준 |
|---|---|---|
| 경기 결과 | 가능 | 최종 경기 결과 예측이므로 수정 가능 |
| 최종 스코어 | 가능 | 후반전 결과를 포함하는 최종 예측이므로 수정 가능 |
| 총 득점 수 | 가능 | 최종 스코어 수정 시 자동 변경 가능 |
| 카드 수 예측 | 가능 | 경기 종료까지 누적되는 항목이므로 수정 가능 |
| 대한민국 첫 득점자 | 조건부 가능 | 전반에 대한민국 득점이 없었을 때만 수정 가능 |
| 첫 골 팀 | 조건부 가능 | 전반에 첫 골이 없었을 때만 수정 가능 |
| 첫 골 시간대 | 조건부 가능 | 전반에 첫 골이 없었을 때만 수정 가능 |
| 전반전 결과 | 불가 | 전반 종료 시 이미 확정된 항목 |
| MVP 예측 | 하프타임 수정 불필요 | 경기 후 최종 수정 제출로 처리 |

### 6.2 하프타임 수정 원칙

- 하프타임 수정은 1인 1회만 허용한다.
- 후반 시작 전까지 제출해야 한다.
- 수정하지 않은 항목은 경기 전 1차 예측값을 그대로 사용한다.
- 전반에 이미 확정된 이벤트는 수정할 수 없다.
- 하프타임 수정 제출 마감 이후에는 MVP를 제외한 모든 항목을 수정할 수 없다.

- 현재 1~3등 중간 순위
- 개인별 현재 중간 점수

- 첫 골 팀 적중자 수
- 첫 골 시간대 적중자 수

### 8.8 카드 수 예측, 5점

카드는 양 팀 합산 경고 수를 기준으로 한다. 레드카드는 별도 처리하지 않고, 운영 편의상 공식 경기 기록 기준 카드 수로 계산한다.

| 선택지 | 설명 |
|---|---|
| 0~2장 | 카드가 적은 경기 |
| 3~5장 | 일반적인 수준 |
| 6장 이상 | 거친 경기 |

| 조건 | 점수 |
|---|---:|
| 카드 수 구간 정확히 일치 | 5점 |

### 8.9 MVP 예측, 5점

MVP는 경기 시작 전에 1차로 입력한다. 다만 MVP는 경기 흐름을 본 뒤 판단하는 재미 요소가 있으므로, 경기 종료 후 MVP 항목만 최종 1회 수정 제출할 수 있다.

| 조건 | 점수 |
|---|---:|
| 경기 후 최종 제출한 MVP가 운영진 선정 MVP와 일치 | 5점 |
| MVP 최종 제출을 하지 않았으나 경기 전 입력값이 운영진 선정 MVP와 일치 | 5점 |
| MVP 예측 불일치 | 0점 |

MVP 선정 기준은 아래 순서로 적용한다.

1. 공식 경기 MOM 또는 POTM이 확인 가능한 경우 공식 기준 적용
2. 공식 기준 확인이 어려운 경우 현장 투표로 MVP 선정
3. 현장 투표가 어려운 경우 운영진이 경기 기여도, 득점, 도움, 경기 영향도를 기준으로 선정

---

## 9. 동점자 처리 기준

동점자가 발생할 경우 아래 순서대로 순위를 결정한다.

1. 정확한 스코어 점수가 높은 사람
2. 경기 결과를 맞힌 사람
3. 대한민국 첫 득점자를 맞힌 사람
4. 첫 골 시간대를 맞힌 사람
5. 하프타임 수정 없이 경기 전 예측값으로 더 높은 점수를 받은 사람
6. 예측 제출 시간이 빠른 사람
7. 그래도 동점이면 공동 순위 또는 현장 가위바위보

웹페이지 구현 시에는 1~6번까지 자동 처리하고, 최종 동점자는 공동 순위로 표시하는 방식을 추천한다.

## 10.3 하프타임 수정 화면

전반 종료 후 운영자가 전반 결과를 입력하면 활성화되는 화면이다.

### 표시 정보

| 영역 | 내용 |
|---|---|
| 전반 실제 결과 | 전반 스코어, 전반전 결과, 첫 골 정보 |
| 내 중간 점수 | 전반 기준 확정 점수 |
| 전체 중간 순위 | 현재 1~3등 또는 전체 순위 |
| 수정 가능 항목 | 경기 결과, 최종 스코어, 카드 수 등 |
| 수정 마감 안내 | 후반 시작 전까지 |

### 수정 입력 항목

| 필드명 | 수정 가능 기준 |
|---|---|
| 경기 결과 | 항상 가능 |
| 대한민국 최종 스코어 | 항상 가능 |
| 멕시코 최종 스코어 | 항상 가능 |

| 카드 수 구간 | 항상 가능 |

| 첫 골 팀 | 전반에 첫 골이 없었을 때만 가능 |
| 첫 골 시간대 | 전반에 첫 골이 없었을 때만 가능 |

### UX 권장 사항

- 수정 가능한 항목과 불가능한 항목을 명확히 구분
- 이미 확정된 항목은 잠금 표시
- 하프타임 수정은 1회만 허용
- 수정 완료 후 변경 전, 변경 후 비교 화면 제공

## 10.4 MVP 최종 수정 화면

경기 종료 후 열리는 참여자용 화면이다. 경기 종료 전에는 노출하지 않거나 비활성화한다.

### 입력 항목

| 필드명 | 입력 방식 | 필수 여부 |
|---|---|---|
| 이름 또는 참여자 확인 | 자동 표시 또는 선택 | 필수 |
| 경기 전 MVP 1차 예측 | 읽기 전용 표시 | 필수 |
| MVP 최종 예측 | 드롭다운 또는 텍스트 입력 | 필수 |
| 선정 이유 | 텍스트 입력 | 선택 |

### UX 권장 사항

- 경기 종료 전에는 최종 제출 버튼 비활성화
- 1인 1회만 수정 제출 허용
- 미제출 시 경기 전 MVP 1차 예측값을 최종값으로 간주
- 순위 발표 전 MVP 최종 제출 마감 시간 명시

## 10.5 예측 현황 화면

경기 전에는 다른 사람의 상세 예측을 모두 공개하지 않는 것을 추천한다. 단, 참여자 수와 일부 통계는 공개하면 재미를 높일 수 있다.

### 공개 추천 정보

| 항목 | 공개 여부 | 설명 |
|---|---|---|
| 전체 참여자 수 | 공개 | 참여 유도 효과 |
| 대한민국 승 예측 비율 | 공개 | 분위기 형성 |
| 무승부 예측 비율 | 공개 | 재미 요소 |
| 멕시코 승 예측 비율 | 공개 | 재미 요소 |
| 가장 많이 나온 스코어 | 공개 가능 | 경기 전 흥미 요소 |
| 개인별 상세 예측 | 경기 전 비공개 추천 | 전략성 유지 |

## 13. 특별상 아이디어

| 특별상 | 선정 기준 |
|---|---|
| 축구도사상 | 최고점자 |

| 아깝상 | 1점 또는 근소한 차이로 순위권 밖인 사람 |
| 응원상 | 가장 열정적으로 응원한 사람 |
| 반전예측상 | 가장 의외의 예측을 했지만 일부 적중한 사람 |
| 스코어장인상 | 정확한 스코어를 맞힌 사람 |
| 예언자상 | 첫 골 팀과 첫 골 시간대를 모두 맞힌 사람 |

특별상은 점수와 별개로 운영하되, 너무 많은 상품을 준비하기보다 1~2개 정도만 추가하는 것을 추천한다.

## 15. 점수 계산 로직 개요

### 15.1 최종 채점 기준값 결정

```text
최종 채점 기준 예측값 =
  하프타임 수정 제출값이 있는 항목은 하프타임 수정값 사용
  하프타임 수정 제출값이 없는 항목은 경기 전 1차 예측값 사용
  전반에 이미 확정된 잠금 항목은 경기 전 1차 예측값 사용
  MVP는 경기 후 최종 제출값 사용
  MVP 최종 제출값이 없으면 경기 전 MVP 1차 예측값 사용
```

### 15.2 총점 계산

```text
총점 = 경기 결과 점수
     + 정확한 스코어 점수
     + 총 득점 수 점수
     + 대한민국 첫 득점자 점수
     + 첫 골 시간대 점수
     + 첫 골 팀 점수
     + 전반전 결과 점수
     + 카드 수 점수
     + MVP 점수
```

### 15.3 중간 점수 계산

```text
전반 중간 점수 = 전반전 결과 점수
              + 전반에 확정된 첫 골 팀 점수
              + 전반에 확정된 첫 골 시간대 점수
              + 전반에 확정된 대한민국 첫 득점자 점수
```

### 15.4 구현 시 유의사항

- 스코어 항목은 부분 점수가 있으므로 별도 함수로 구현
- 첫 골 시간대는 인접 구간 점수 처리를 위해 순서값을 부여
- 대한민국 첫 득점자는 무득점 시 “없음” 처리를 명확히 해야 함
- 경기 시작 후 1차 예측 제출 마감 처리 필요
- 전반 결과 입력 후 하프타임 수정 상태로 전환 필요
- 하프타임 수정은 1인 1회만 허용
- 전반에 확정된 항목은 수정 불가 처리 필요
- MVP 최종 수정은 경기 종료 후에만 가능하도록 상태값 분리
- 동점자 처리 기준을 정렬 로직에 반영

## 19. 권장 운영 원칙

- 금전 베팅 요소는 제외한다.
- 참여자는 경기 시작 전까지만 1차 예측을 제출할 수 있다.
- 전반 종료 후 중간 점수를 공개하되, 최종 순위가 아님을 명확히 안내한다.
- 하프타임 수정은 일부 항목에 한해 1회만 허용한다.
- 전반에 이미 확정된 항목은 수정할 수 없다.
- MVP 최종 수정은 경기 종료 후부터 순위 발표 전까지만 진행한다.
- 점수 기준과 수정 가능 항목은 사전에 모두 공개한다.
- 운영자는 전반 결과와 최종 경기 결과를 공식 기록 또는 현장 확인 기준으로 입력한다.
- 동점자 처리 기준을 사전에 명확히 공지한다.
- 웹페이지에는 “친목 목적의 예측 게임”이라는 안내 문구를 명시한다.

## 20. 최종 추천안

본 이벤트는 아래 방식으로 운영하는 것을 추천한다.

```text
방식: A안, 실력형 순위제
총점: 100점 만점
제출 구조: 경기 전 1차 예측, 하프타임 일부 항목 수정, 경기 후 MVP 최종 수정
중간 이벤트: 전반 종료 후 중간 점수 공개
선정: 총점 기준 1등, 2등, 3등
동점 처리: 스코어 점수, 경기 결과, 득점자, 첫 골 시간대, 하프타임 수정 여부, 제출 시간 순
운영 채널: 모바일 웹페이지
상품: 1~3등 상품 + 특별상 1~2개
운영 목적: 팀 워크샵 친목 및 경기 관람 몰입도 향상
```

실력형 순위제는 점수 기준이 명확하고 결과 발표가 직관적이라는 장점이 있다. 여기에 전반 종료 후 중간 점수 공개와 하프타임 수정 기회를 추가하면 경기 내내 참여 몰입도를 유지할 수 있다. 다만 공정성을 위해 이미 확정된 항목은 수정하지 못하도록 잠금 처리하고, 수정 가능 항목과 마감 시간을 웹페이지에서 명확히 안내하는 것이 중요하다.
